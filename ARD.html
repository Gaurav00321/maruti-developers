Section 01

Executive Summary & Project Vision

Maruti Developers is a Vadodara-based industrial real estate consultancy (est. 1998) with 25+ years of experience and 2,500+ clients. This platform must position them as the most credible, transparent, and professional industrial property advisor in India — targeting factory promoters, logistics investors, NRIs, and next-gen business families across 7+ states.

Core Business Objectives

Primary Goal

Generate qualified B2B leads — factory promoters, warehouse investors, NRIs — who submit their Name + WhatsApp via the lead magnet popup or contact forms.

Secondary Goal

Establish authority via blog content, verified property listings, and a transparent "7 differentiators" narrative that converts skeptical buyers.

Revenue Impact

Each converted lead can represent ₹5–100 crore deal value. Even 2–3 extra conversions per month at standard commission = ₹30–90L additional annual revenue.

Brand Authority

Pan-India positioning from a Gujarat HQ requires extreme credibility signals: state coverage maps, verified listings, RERA compliance badges, founder story.

  • Lead conversion rate ≥ 3.5% of organic traffic
  • Popup lead capture rate ≥ 12% of visitors who see it
  • WhatsApp contact rate ≥ 60% of all leads submitted

  • Number counters: countUp animation on scroll (2500 clients, 25 years)
  • Property cards: translateY(-4px) + border-color glow on hover
  • Navbar: blur + bg opacity transition on scroll, height compress

Lead Magnet Popup

The popup is the #1 conversion tool. It fires on: (1) 30-second time delay, (2) exit-intent mouseout, (3) 60% scroll depth. It shows ONCE per session (localStorage flag) and re-shows after 7 days.

  • Trigger: 30s delay OR exit intent OR 60% scroll (whichever first)
  • Fields: Full Name (text), WhatsApp Number (tel), Industry (select), Preferred State (select)
  • CTA: "Get My Free Plot Shortlist" → submit to /api/leads
  • Post-submit: Thank you screen with WhatsApp deep-link button
  • Dismiss: X button, Escape key, overlay click — sets 7-day localStorage cookie
  • Mobile: Full-screen drawer from bottom (not floating card)
  • Accessibility: Focus trap, aria-modal, keyboard nav

Properties Page ( /properties )

  • Filter sidebar/top bar: State, City, Type (Industrial/Warehouse/Commercial), Size range, Budget range, Available for (Buy/Lease/Rent)
  • Property card: Image carousel, headline, size (sq ft/acres), price/rate, state badge, CTA button
  • URL-syncable filters (shareable links): ?state=Gujarat&type=warehouse&budget=5cr
  • Infinite scroll OR pagination (12 per page)
  • Map view toggle (Leaflet)
  • Sort by: Latest, Price Low-High, Size
  • No-results state with "Request Custom Shortlist" CTA

// Response 201:

// Side effects:
// 1. Supabase INSERT into leads table
// 2. Resend.sendEmail() → admin notification with all lead details
// 3. Resend.sendEmail() → lead confirmation with WhatsApp link

WhatsApp Integration Strategy

  • All form submits generate a wa.me pre-filled deep link shown to the user post-submit
  • Admin receives WhatsApp notification via Twilio or Wati webhook (Phase 2)
  • Click-to-WhatsApp buttons on every CTA section use https://wa.me/919898610678
  • Floating WhatsApp bubble (bottom-right, fixed position) on all pages
  • Mobile: opens WhatsApp app directly. Desktop: opens web.whatsapp.com

Supabase Filter Query Pattern


let query = supabase
  .from('properties')
  .select('*')
  .eq('status', 'active')

if (filters.state) query = query.in('state', filters.state)
if (filters.type) query = query.eq('type', filters.type)
if (filters.transaction) query = query.eq('transaction', filters.transaction)
if (filters.budget_min) query = query.gte('price', filters.budget_min)
if (filters.budget_max) query = query.lte('price', filters.budget_max)
if (filters.size_min) query = query.gte('size_sqft', filters.size_min)
if (filters.railway_siding) query = query.eq('railway_siding', true)
if (filters.pollution_cat) query = query.in('pollution_cat', filters.pollution_cat)

query = query
  .order(sortBy, { ascending: sortOrder === 'asc' })
  .range(page * 12, (page + 1) * 12 - 1)
